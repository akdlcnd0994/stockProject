# 한국증시 대시보드

#### 프로젝트 링크
https://akdlcnd0994.github.io/stockProject

#### 프로젝트 설명
클로드 코드를 활용하여 한국증시의 금일 등락률 및 시가총액 순위를 간결하게 보여주는 프로젝트

1. 한국투자증권 API를 활용해 KOSPI/KOSDAQ 지수, 상승/하락 종목, 시가총액 상위 종목 등 실시간 주식 데이터를 수집하는 서비스
2. Spring Boot 백엔드가 주기적으로 주가 데이터를 DB에 저장하고 REST API로 제공하며, React 프론트엔드에서 차트로 시각화
3. 종목별 일봉/주봉/월봉 캔들 차트와 거래량 차트를 Recharts로 렌더링하는 주식 대시보드 웹앱

<br/>

#### 개발 환경 및 언어
backend
- Spring Boot(Intellij)
- MySQL(Mybatis)

frontend
- React
